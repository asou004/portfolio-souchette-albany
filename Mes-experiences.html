Mes expèriences

Découvrez certains de mes projets...

  • 1- Développement web en Node JS: Zoom Clone

    Le but de cet exercice va être de créer le clone de l'application Zoom avec Node JS à l'aide du tutoriel "Build a Zoom Clone with Node JS for Beginners" diffusé le 05 août 2020 (https://youtu.be/ZVznzY7EjuY). Cette application permet de se voir et de chatter à distance avec n'importe qui même avec soi-même en utilisant plusieurs pages de navigateur en rafraîchissant la page à chaque connexion d'une personne. Technologies utilisées sont le EJS qui est un langage de création de modèles simple qui vous permet de générer un balisage HTML avec du JavaScript brut. CSS3, Bootstrap, Font Awesome et JavaScript ES6 pour les langages. Les frameworks utilisés sont Node JS, Socket.io pour émettre les connexions à une "Room", Peer Server pour recevoir les connexions,Git Hub pour sauvegarder les fichiers à distance et Heroku pour déployer l'application.

  • 2- Développement web en React JS: Netflix Clone

    Le but de ce projet a été de créer un clone de Netflix en React JS avec le tutoriel "How to Build a NETFLIX Clone with REACT JS for Beginners" diffusé le 09 juillet 2020 (https://youtu.be/XtMThy8QKqU). Cette application permet de voir des bandes annonces de films sur Youtube. Rafraîchir la page pour réinitialiser les contenus. Technologies utilisées sont le HTML5, CSS3 et JavaScript ES6 pour les languages. Les frameworks utilisés sont React JS pour le front-end, TMDB pour l'obtention de la clé API des bandes-annonces de films et Firebase pour le déploiement de l'application.

  • 3- Développement web en React JS: Chat

    Le but de ce projet a été de créer un "Chat" au modèle de Facebook en React JS. Je me suis appuyé en autodidacte du tutoriel "Build and Deploy a Realtime Chat Application - Socket.io, Node.js, and React.js" (https://youtu.be/ZwFA3YMfkoc) Technologies utilisées sont le HTML5, CSS3, JavaScript ES6 avec le framework React JS qui utilise la technologie Node JS et Netlify et Heroku afin d'héberger la page Front-End et le Back-end. Entrer un nom et une salle de votre choix.

  • 4- Développement web en Vue JS: Wesh

    Le but de cet exercice va être de créer une application responsive E-commerce avec Vue JS en utilisant le tutoriel "https://www.youtube.com/results?search_query=projet+vueJs" Technologies utilisées sont le HTML5, CSS3, JavaScript avec le framework JavaScript évolutif " Vue Js " en lien avec Node JS. J'ai appris en autodidacte à l'aide de tutoriels où j'y travaille un peu plus chaque jour.

  • 5- Animation publicitaire: ProAttitude.com

    Le but de cet exercice va être de créer un site web vitrine d'un bar à smoothies. Cette interface contient du HTML5, CSS3 principalement, Boostrap mais ne contient pas de JavaScript, la map est une image statique.
