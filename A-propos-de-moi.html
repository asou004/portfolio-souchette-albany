BIO

Je suis développeur Web et d'application mobiles (autodidacte,formation et stage). Les deux principales composantes de mon travail sont le développement Front-end de sites web et la réalisation d'applications mobiles en React JS et Back-end avev PHP, MySQL, JAVA, Android studio débutant.
Cela fait six ans que j'ai découvert le développement informatique puis je m'y suis converti professionnellement à cette activité jusqu'à ce jour. Je suis actuellement à la recherche de nouveaux projets afin d'apporter mes acquis aux futurs sites mais aussi en apprendre d'autres.

Développeur Web Front-End

Je développe toute l'interface visible de votre site Web à partir d'une maquette. Le but étant d'aboutir à une interface ergonomique et interactive.

Technologies utilisées : HTML5, CSS3, JavaScript, ReactJS.

Développeur Web Back-End

Je réponds à un cahier des charges en développant toutes les fonctionnalités invisibles à l'utilisateur et nécessaires au fonctionnement de l'interface.

Technologies utilisées : SQL-PhpMyadmin, PHP-Symfony, Java-Eclipse, Android Studio

WordPress

J'utilise ce CMS afin de créer un site Web entièrement modulable par le client à l'aide d'un back office intuitif.
